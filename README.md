# Concept Trainer
Add a concept, write an explanation, and test yourself with quick quizzes.

## Features
- Create subjects and concepts with short explanations
- Practice by matching explanations to the right concept
- Light/Dark theme toggle

## Tech Stack
- React + Vite
- Firebase Auth
- Cloud Firestore
- Firebase Hosting

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Create a `.env.local` file using `.env.example` as a template.
3. Run the app:
   ```bash
   npm run dev
   ```

## Build
```bash
npm run build
```

## Deploy (Firebase Hosting)
```bash
firebase deploy
```
